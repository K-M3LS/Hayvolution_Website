Via l'interface, il est possible de créer sa propre espèce afin de l'ajouter à la simulation et expérimenter les résultats de cet ajout. Voilà les statistiques pouvant être définies :



— un nom d’espèce pour pouvoir les différencier entre eux.
— une alimentation, pour définir si l'espèce est herbivore ou carnivore.
— une vitesse de déplacement, qui définira le nombre de cases sur lesquelles un individu de l'espèce peut se déplacer.
— une perception qui permet à l’animal de voir les objets autour de lui dans un certain rayon.
— une force qui leur sera utile lors du combat.
— un taux de nourriture ainsi qu’une valeur de nourriture.
— un taux de reproduction pour définir quand l'espèce peut se reproduire ou non en fonction de sa nourriture.
